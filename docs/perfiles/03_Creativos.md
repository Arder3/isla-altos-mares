# Estrategia de Perfil: CREATIVOS (Pipeline y Activos)

## 🎯 Objetivo
Asegurar la coherencia técnica, el cumplimiento de estándares industriales y la eficiencia en la producción de activos.

## 🔓 Niveles de Acceso (IDs)
| ID | Descripción | Propósito |
| :--- | :--- | :--- |
| **01 - 14** | Todos los Activos | Control total de la evolución del personaje. |
| **04** | Material HD | Acceso a archivos fuente con máxima resolución. |
| **-** | Pipeline de Arte | Consulta de manuales de nomenclaturas y flujos. |
| **-** | Registros SSoT | Validación de metadatos y IDs únicos. |

## 🧭 Navegación y UX
- **Flujo**: Galería tipo "Asset Manager" con visualización de mallas y hojas técnicas.
- **Interacción**: Filtros técnicos avanzados (Resolución, Estado de Aprobación, Fecha).
- **Tono**: Profesional, eficiente y estandarizado.

## 🌟 Puntos Fuertes (Strengths)
- Estandarización industrial (Universal Image ID Rule).
- Sincronización automática "Zero-Touch" entre Drive y Cloudinary.
- Nomenclatura Pascal_Underscore consistente.

## ⚠️ Análisis de Gaps (Debilidades)
- **Categorías Incompletas**: El portal solo muestra Personajes. Faltan Locaciones, Utilería y Vehículos.
- **Visualizador 3D**: No existe integración para ver modelos 3D si se llegaran a producir.
- **Gestión Directa**: El creativo no puede "aprobar" o "rechazar" un archivo desde el portal (solo lectura).
