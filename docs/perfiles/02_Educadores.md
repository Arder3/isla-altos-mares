# Estrategia de Perfil: EDUCADORES (Material Didáctico)

## 🎯 Objetivo
Proporcionar una base narrativa y emocional sólida que permita utilizar el universo de "Isla de Altos Mares" como herramienta pedagógica.

## 🔓 Niveles de Acceso (IDs)
| ID | Descripción | Propósito |
| :--- | :--- | :--- |
| **01** | Concepto Maestro | Identificación de personajes clave. |
| **12** | Hojas de Expresión | Análisis de emociones y respuestas afectivas. |
| **13** | Catálogo de Gestos | Estudio del lenguaje no verbal y personalidad. |
| **-** | Fichas Narrativas | Acceso a las motivaciones y arcos de personaje (Biblia). |

## 🧭 Navegación y UX
- **Flujo**: Galería estructurada con descripciones textuales de los arcos narrativos.
- **Interacción**: Filtros por "Emoción" o "Rasgo de Personalidad".
- **Tono**: Formativo, detallado y reflexivo.

## 🌟 Puntos Fuertes (Strengths)
- Espectro emocional muy detallado en el Catálogo de Expresiones.
- Coherencia psicológica de los personajes (Mita como líder reguladora, Cito como motor de cambio).

## ⚠️ Análisis de Gaps (Debilidades)
- **Guías de Uso**: No existen "Guías del Docente" que sugieran cómo usar estos activos en clase.
- **Taxonomía**: Falta una clasificación de los personajes por "Valores Sugeridos" (ej. Empatía, Liderazgo).
- **Accesibilidad**: No hay herramientas de lectura en voz alta o descripciones alternativas para inclusión.
